== Description ==

Automatically generate short link url when publishing post.
发布文章时自动生成短网址URL。

== Installation ==

1. Upload `qqworld-short-url` directory to `/wp-content/plugins/` directory
上传 `qqworld-short-url` 目录到 `/wp-content/plugins/` 文件夹
2. Activate the plugin through the 'Plugins' menu in WordPress
在wordpress的 '插件' 菜单中激活该插件
